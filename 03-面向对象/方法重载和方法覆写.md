## 一、 方法重载（Overload）示例

### 核心特点：

1. **同一个类中**
2. 方法名**相同**
3. 参数列表**不同**（参数个数、类型、顺序不同，与返回值无关）
4. 是编译时多态

### 简单案例（计算两个 / 三个数的和）


```java
/**
 * 方法重载示例：同一个类中，方法名相同，参数列表不同
 */
public class OverloadDemo {

    // 方法1：计算两个int类型的和
    public static int add(int a, int b) {
        System.out.println("两个int参数的add方法");
        return a + b;
    }

    // 方法2：计算三个int类型的和（参数个数不同）→ 重载
    public static int add(int a, int b, int c) {
        System.out.println("三个int参数的add方法");
        return a + b + c;
    }

    // 方法3：计算两个double类型的和（参数类型不同）→ 重载
    public static double add(double a, double b) {
        System.out.println("两个double参数的add方法");
        return a + b;
    }

    public static void main(String[] args) {
        // 调用时，编译器根据参数列表自动匹配对应的方法
        int sum1 = add(1, 2);
        int sum2 = add(1, 2, 3);
        double sum3 = add(1.5, 2.5);

        System.out.println("sum1 = " + sum1);
        System.out.println("sum2 = " + sum2);
        System.out.println("sum3 = " + sum3);
    }
}
```

### 运行结果：

plaintext

```
两个int参数的add方法
三个int参数的add方法
两个double参数的add方法
sum1 = 3
sum2 = 6
sum3 = 4.0
```

## 二、 方法覆写（Override / 重写）示例

### 核心特点：

1. **父子类之间**
2. 方法名**相同**
3. 参数列表**完全相同**（个数、类型、顺序一致）
4. 返回值类型**兼容**（子类返回值可以是父类返回值的子类，基本类型必须完全相同）
5. 访问权限**不能更严格**（父类`public`，子类不能是`private`/`protected`）
6. 是运行时多态
7. 常用`@Override`注解标记，提高代码可读性并让编译器做校验

### 简单案例（动物叫的行为重写）

java

运行

```
/**
 * 方法覆写示例：父子类之间，方法名、参数列表完全相同
 */
// 父类：动物
class Animal {
    // 父类方法：动物叫
    public void makeSound() {
        System.out.println("动物发出了某种声音");
    }
}

// 子类：狗（继承自动物）
class Dog extends Animal {
    // 子类方法：重写父类的makeSound方法（狗叫）
    @Override // 注解标记这是一个重写方法，编译器会校验正确性
    public void makeSound() {
        System.out.println("小狗汪汪汪叫");
    }
}

// 测试类
public class OverrideDemo {
    public static void main(String[] args) {
        // 父类引用指向子类对象（多态体现）
        Animal animal1 = new Animal();
        Animal animal2 = new Dog(); // 向上转型

        // 调用方法：运行时根据实际对象类型决定调用哪个方法
        animal1.makeSound(); // 调用父类的方法
        animal2.makeSound(); // 调用子类重写后的方法
    }
}
```

### 运行结果：

plaintext

```
动物发出了某种声音
小狗汪汪汪叫
```

## 三、 核心差异总结（快速区分）

|特性|方法重载（Overload）|方法覆写（Override）|
|---|---|---|
|存在范围|同一个类中|父子类之间|
|方法名|相同|相同|
|参数列表|不同（个数 / 类型 / 顺序）|完全相同|
|返回值类型|无强制要求（可不同）|兼容（基本类型相同，引用类型子类可更具体）|
|访问权限|无强制要求|子类不能比父类更严格|
|多态类型|编译时多态（编译器匹配）|运行时多态（运行时根据对象类型匹配）|
|注解|无专用注解|推荐使用`@Override`注解|

### 总结

1. 方法重载是**同一个类内的 “同名不同参”**，方便调用者用统一的方法名操作不同参数的场景。
2. 方法覆写是**父子类间的 “同名同参”**，用于子类重写父类的默认行为，体现多态。
3. 重载看 “参数列表”，覆写看 “父子类 + 同名同参”，这是两者最核心的区分点。
